## 1. 서비스 개요

**서비스명:** Triply *(트리플리)*

**한줄 소개:** 사진 한 장으로 완성되는 나만의 여행 아카이브

**핵심 가치**
여행의 순간은 찰나지만, 기록은 영원해야 한다.
복잡한 여행 일지 대신 — 찍고, 올리고, 자동으로 쌓인다.

**타겟 사용자**
- 20~30대 여성
- 여행을 자주 다니지만 기록은 늘 미루게 되는 사람
- 인스타에 올리는 것과 별개로 "나만의 공간"에 조용히 모아두고 싶은 사람

---

## 2. 문제 정의

**기존 서비스의 불편함**

| 서비스 | 불편함 |
|--------|--------|
| 네이버 블로그 / 노션 | 직접 글 쓰고 사진 정리해야 해서 귀찮음 |
| 인스타그램 | 공개 피드 중심이라 개인 아카이브로 쓰기 어색함 |
| 구글 타임라인 | 기록은 되지만 감성 없고, 사진과 연결이 안 됨 |

**우리가 해결하는 문제**
- 여행 후 기록해야 한다는 심리적 부담 제거
- 내가 언제, 어디서, 무엇을 했는지 자동으로 정리
- "그때 갔던 그 카페 어디였지?"를 바로 찾을 수 있게

## 3. 시장 조사

**유사 서비스 분석**

| 서비스 | 특징 | 한계 |
|--------|------|------|
| Google Timeline | 자동 위치 기록 | 감성 없음, 사진 연동 약함 |
| Polarsteps | 지도 우선 여행 트래킹, 2025년 기준 1,500만 명 유저 | 영문 서비스, UI 복잡 |
| TripMemo | 감성 아카이브, TripBook 개념, 빈티지 필터 | 신규 서비스, 인지도 낮음 |
| Day One | 감성 저널링 교과서, 자동 메타데이터 | 여행 특화 기능 없음 |
| 트리플 | 여행 계획 + 후기 | 기록 아카이브 기능 약함 |

**Triply가 채우는 틈새**
자동화 + 감성 아카이브를 동시에 잡는 국내 서비스가 없음.
사진 기반 자동 기록 + 감성적 기록 카드 뷰의 조합.

**레퍼런스**
- Polarsteps (자동 여행 트래킹, 지도 우선 UI): https://www.polarsteps.com
- TripMemo (감성 여행 아카이브, TripBook): https://tripmemo.app
- Day One (감성 저널링 UI/UX): https://dayoneapp.com
- 1 Second Everyday (최소 입력 → 최대 감동 구조): https://1se.co
- Lemon8 (20~30대 여성 감성 스크랩북 트렌드): https://www.lemon8-app.com

## 4. 핵심 기능 정의

**주요 기능 목록**

1. **사진 기반 자동 기록** — 사진 업로드 시 exif 데이터로 날짜·위치 자동 파싱
2. **기록 카드 뷰** — 중복 장소 제외 후 사진별 1장씩 슬라이드 카드 자동 생성 (감성 키워드 태그, 사진, 장소, 날짜, 날씨, 스티커 포함 / 마지막 장에 지도 표기)
3. **장소 검색** — 내가 방문했던 곳을 키워드·태그·지역으로 검색
4. **지도 뷰** — 카카오맵 위에 내 방문 핀 전체 시각화
5. **소셜 로그인** — 구글 간편 로그인 (MVP), 카카오 로그인 (2차)

**MVP (1주일 범위)**
- 구글 소셜 로그인
- 사진 업로드 → exif 파싱 → 기록 카드 자동 생성
- 홈 최근 여행 슬라이드 + 더보기(시간순 / 지역별 / 카테고리별 목록)
- 방문 장소 키워드 검색

> **MVP 이후 2차 범위:** 카카오 로그인, 날씨 자동 연동, 스티커 커스텀, 지도 뷰 고도화, PDF·포토북 내보내기
