인스타에 올리는 것과 별개로, 나만 보는 공간에 여행 기억을 조용히 모아두고 싶은 분들.

Problem

기존 서비스의 불편함

왜 지금까지 마음에 드는 여행 기록 앱이 없었을까요?

😩

기록해야 한다는 부담

여행 후 직접 글 쓰고 사진 정리하는 과정이 귀찮고, 결국 아무것도 안 하게 됩니다.

📢

공개 피드 중심 SNS

인스타그램은 타인의 시선이 의식되어 개인 아카이브 용도로 사용하기 어색합니다.

🗂️

감성 없는 자동 기록

구글 타임라인은 위치 기록은 되지만 사진과의 연결이 없고, 추억을 되살리기 어렵습니다.

발걸음이 채우는 틈새

자동화 + 감성 아카이브를 동시에 잡는 국내 서비스가 없습니다.
사진 기반 자동 기록 + 감성적 기록 카드 뷰의 조합으로, 기록 부담 없이 나만의 여행 서랍을 완성합니다.

Core Features

핵심 기능 5가지

사진 한 장이면 충분해요. 나머지는 발걸음가 다 해드립니다.

FEATURE 01 📷

사진 기반 자동 기록

사진을 올리면 EXIF 데이터로 촬영 날짜와 위치를 자동으로 읽어내요. 손가락 하나 안 움직여도 기록이 완성됩니다.

MVP 포함
FEATURE 02 📖

감성 기록 카드 뷰

도트 격자 속지 위에 사진, 장소명, 날씨, 감성 태그, 스티커가 자동으로 배치돼요. 마지막 장엔 지도가 펼쳐집니다.

MVP 포함
FEATURE 03 🔍

장소 검색

"그때 갔던 그 카페 어디였지?" — 키워드, 태그, 지역으로 내 방문 기록을 바로 찾을 수 있어요.

MVP 포함
FEATURE 04 🗺️

지도 뷰

카카오맵 위에 내 모든 방문 핀이 시각화돼요. 내가 얼마나 많은 곳을 다녔는지 한눈에 볼 수 있어요.

2차 고도화
FEATURE 05 🔐

소셜 로그인

구글 계정으로 간편하게 시작하세요. 내 데이터는 나만 볼 수 있도록 Row Level Security로 보호됩니다.
